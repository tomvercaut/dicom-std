# DICOM-std `test-data`

This subproject implements a library to generate test data by downloading parts of the DICOM standard in XML format. The DICOM standard parts are downloaded while building the crate. The data is stored in the build directory.

This crate is part of the [DICOM-std](https://github.com/tomvercaut/DICOM-std).

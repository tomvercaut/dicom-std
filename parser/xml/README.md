# DICOM-std `parser`

This subproject implements a library to model and parse the DICOM standard in XML format.

The library parses specific parts of the DICOM standard required to build a model of the Information Object Definitions (IODs) and composite IODs. It also builds a data dictionary with all registered DICOM tags.

This crate is part of the [DICOM-std](https://github.com/tomvercaut/DICOM-std).
